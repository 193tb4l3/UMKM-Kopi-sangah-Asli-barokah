Referensi Sumber Kode

  • $(document).ready()
    Digunakan untuk memastikan bahwa seluruh dokumen HTML telah dimuat sebelum menjalankan kode JavaScript.
    jQuery `ready()`
  • offset()
    Mengembalikan posisi saat ini dari elemen relatif terhadap dokumen.
    jQuery `offset()`
  • scrollTop()
    Mengembalikan atau mengatur posisi scroll vertikal dari elemen.
    jQuery `scrollTop()`
  • on()
    Menambahkan penanganan acara untuk satu atau lebih peristiwa ke elemen yang dipilih.
    jQuery `on()`
  • trigger()
    Menjalankan peristiwa tertentu untuk elemen yang dipilih.
    jQuery `trigger()`
  • Bootstrap CSS
    Digunakan untuk menambahkan gaya dan layout yang responsif pada halaman web.
    Bootstrap CSS
  • Animasi CSS dengan @keyframes
    Digunakan untuk mendefinisikan animasi melalui CSS dengan menetapkan langkah-langkah perubahan gaya dari awal hingga akhir.
    W3Schools - CSS Animation
